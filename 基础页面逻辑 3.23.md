# （一）从零开始的新建商品目录的完整流程：

1.新建供应商（建立供应商字段，此时供应商字段在系统内为：不可用）→上传营业执照/经营许可证（完成后供应商字段状态在系统内为：可用）
供应商名称具有唯一性

2.为供应商建立注册证（重要），并提交附件保存（注册证状态变为可用）。
注册证不具有唯一性。统一注册证名称甚至可以出现在不同供应商下方

3.此时就可以在创建商品目录时，选择供应商、选择并绑定注册证，之后继续手动录入其他商品信息，保存明细，提交商品信息，系统生成商品目录唯一码（商品编码），该商品目录在系统内可用，完成商品目录建立流程。（创建的商品明细呈现在字典维护/物资字典内）

树状图层级关系：

供应商A
   注册证a
      商品1a
      商品2a
      ...
   注册证b
      商品1b
      商品2b
      ...
      ...

可以理解为要先新建一个“供应商”一级文件夹空，在下面再建数个“注册证”二级文件夹，最后在新建文件“商品明细”记录时，可以选择要储存到哪里：“/供应商A/注册证b/”，最后点击保存，系统为这个文件生成唯一的“识别码”——商品编码（不是乱序或随机生成，是按照统一格式、并且按照新增顺序一一发放。）→完成商品明细记录创建。
商品目录全部呈现在“物资字典”内，可执行搜索、修改、删除等操作，但是后续需要做出修改限制。

注意：同一个商品明细记录不可以重复绑定不同供应商/注册证，具有唯一性。
同一个供应商可以拥有多个注册证（多少个目录就有多少个注册证）



# （二）院区建立/科室建立/账户建立
同上，院区为一级目录，院区下科室为二级目录，这里先从单一院区开始：
默认院区为初始院区，对应需求当中的查看所有院区库存、目录等操作的需求（如果不需要也能实现则忽略这一条）

（默认为空院区）新建院区（输入文件夹名称）→→→新建科室（在院区文件夹下面再建几个文件夹，输入你想要的名称，以后它就叫“科室”了）→→→完成科室和院区创建

创建新账号（新建一个名叫“张三”的文件）→→→选择科室（你要把它放哪个文件夹下面，它就属于哪个科室的，拥有可操作那个科室的权限。如何划分不同人员的权限：将操作权限写在“张三”的文件内。要同时拥有不同科室的操作权限，只需要把“张三”文件拖到更上一级的目录即可，或者在每一个“科室”文件夹内都放一个“张三”的文件）→→→完成账号创建

角色模板：创建角色时，选择所属科室后，要给他什么访问权限，限定他可以操作的内容，提前设定好模板之后，就可以一键添加权限。（包括单一权限或页面的访问权限，需要手动详细设定/或者同时勾选数个角色模板）


# （三）商品采购、入库、出库、消耗、结算数据全流程闭环

开始之前，默认已经确认有可用供应商条目、完善注册证、已经建立好商品目录

订单状态分为（待提交、待审核/已驳回、待收货、待入库、已完成）

采购单据完成提交步骤进入（待审核）状态时，就会生成一个对应的【采购单号】


1、采购

由科室发起采购申请（进入采购申请界面、获取全院商品目录，筛选搜索条件并设定需要采购的商品，录入需要采购的数量，提交单据）→采购单据进入科室采购审核（由科室负责人审核或指定权限人），可进行通过、修改采购订单、驳回三种操作，通过采购申请后，采购单据进入到【采购收货】页面


2.采购收货
对已经完成采购审核的订单进行确认收货操作，在这里需要核对[实际到货的数量]填入并完成收货，订单完成收货后，进入【验收入库】详细商品信息录入流程
到货数量与采购数量一致，确认收货即可；
实际到货数量因为各种原因不足采购数量时，在备注栏手动填写原因，提交后将按照[实际填写数量]提交订单至[验收入库]，不足采购数量的部分（或未验收物资明细），将按照原始单据号推送采购申请记录到【异常订单管理】，已完成收货明细单据进入[采购入库]页面

3.【验收入库】
完成收货的订单，将会进入【验收入库】页面。[查单据]界面显示所有已完成收货的采购单，点击对应的采购单之后，就可以录入批号效期等详细信息。

录入信息时，默认只显示数条不重复、唯一商品编码的待录入信息目录。
当实际货物出现可以允许的同一商品不同效期、批号的情况，先修改【入库数量】对应批号货物的数量，双击该条明细新增该商品的新空明细行（就是没有录入批号效期的空白行，左边显示剩余没有入库的采购数量）。
最终实现：采购10箱注射器，5箱、3箱、2箱都是不同批号效期，但是总数量一致。（交单据时不能超出采购数量）


4、消耗出库
有对应操作权限的人即可进行消耗出库操作。撤销消耗由对应科室负责人负责管理

5、结算
按照当月入库数据/消耗数据生成结算数据，并按照供应商合并各个科室数据
结算数据由院区财务部门管理。

6、物资调拨
由科室负责人发起调拨申请，选择急需调拨到耗材，然后可以查看拥有对应耗材的全院库存，选择科室并提交调拨申请。

方案一：由被调拨科室负责人审核，是否同意或拒绝
方案二：由上级管理部门处理调拨申请，并查看全院科室库存，选择调拨科室调拨物资。

物资被调拨后，原始入库数据不变，当月核算成本依旧算在初始入库科室。


7.采购退货【待添加功能】

（1）科室进入采购管理/采购退货页面，选择[新建退货申请]，[选择物资耗材]，上方可快速筛选：选择近效期范围（≤1个月、≤3个月、≤6个月）缩小筛选范围，或者直接搜索对应物资名称或物资编码，添加需要退货的物资明细（不限制供应商，选择多个不同供应商物资后，会将按照供应商将单据拆分）。科室手动填写退货原因：近效期、产品质量问题、滞销等原因，提交退货申请
（2）[科室退货审核]拥有对应审核权限的科室人员进行审核，审核通过后将会从[库存管理/物资库存]中扣除物资库存（删除该条明细记录，但是与该商品明细有关的记录不会删除，仍然可查询）

[折旧类型]
以下为AI回答：
一、 折旧类型的作用
在管理系统中，折旧类型主要有三大作用：财务核算自动化、成本归集精细化、合规审计可追溯。

1. 自动计算月折旧额
系统根据折旧类型，结合资产原值、预计使用年限、残值率，自动生成每个月的折旧凭证。

平均年限法：医院最常用的方法。系统每月扣减固定的金额，确保资产净值在寿命期末归零或达到残值。

工作量法：针对大型设备（如CT、MRI）。系统根据设备当月实际扫描人次或开机小时数计提折旧。如果删除折旧类型，系统将无法自动生成这部分成本数据。

2. 影响科室成本核算
在医院精细化管理中，折旧是全成本核算的重要组成部分。

手术室的无影灯、麻醉机折旧，会计入手术室的直接成本。

放射科的MRI折旧，会计入放射科的直接成本。

如果折旧类型缺失或设置错误，会导致科室成本虚高或虚低，直接影响绩效核算和DIP/DRG（按病种分值付费/疾病诊断相关分组）医保支付背景下的病种成本分析。

3. 满足政府会计制度与审计要求
根据《政府会计制度——行政事业单位会计科目和报表》，医院固定资产必须计提折旧（文物、图书除外）。

审计或财政检查时，会重点核查折旧方法是否一致、是否按规定计提。

折旧类型是判断账务处理是否合规的关键依据。如果删除该字段，系统导出的资产报表将不符合财政要求，可能导致审计调整甚至通报。
